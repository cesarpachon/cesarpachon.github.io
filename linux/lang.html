dictionaries

there are many wrappers for online services like google translate, but there is also the dictd daemon and the dict client:

sudo apt-get install dictd sudo apt-get install dict sudo apt-get install dict-freedict-eng-spa

above commands will install the dict daemon and client, and the english-spanish dictionary.

list the installed dictionaries:

dict -D

translate something:

dict -d fd-eng-spa "case"

default eng-spa is too poor.. but you can add more online dictionaries to the service:

sudo apt-get install dict-gcide sudo apt-get install dict-wn sudo apt-get install dict-devil sudo apt-get install dict-moby-thesaurus
