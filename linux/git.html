Git commands

this one will save your life, if by mistake you delete both the remote and local branch:

git log -g

http://stackoverflow.com/questions/19592275/git-recover-branch-deleted-on-remote-and-local-folder-deleted

cool command line tool for inspect git history: https://github.com/rgburke/grv

grv -repoFilePath ./

checkout a branch:

git checkout BRANCH_NAME

create a new local branch and then upload it to the origin

git checkout -b BRANCH_NAME
git push origin -u BRANCH_NAME

commit only changes in the given branch

git push origin BRANCH_NAME

List all available branches

git branch -a

Update the local repo with the status of all the remote branches

git fetch

Delete a remote branch

git push origin --delete BRANCH_NAME

delete a local branch

git branch -d BRANCH_NAME

Bring down remote branches

git pull

save/restore temporal changes into a local stack

git stash / git stash pop

compare two branches, displaying only the names of the files that changed

git diff --name-status branch1..branch2

compare same file in two branches

git diff branch1 branch2 -- myfile

Compare a file added to stage with version in HEAD.

git diff HEAD -- file
git diff HEAD^^ HEAD file
git diff HEAD~2 HEAD file

See the detailed history and changes of a single file

git log --follow -p FILENAME

merge a specific file (two steps!):

git checkout branch_destiny
git checkout --patch branch_source file

list the commits and comments of the current branch

git log
git log --pretty=oneline
list graph, hash, author, date, subject
git log --pretty=format:"%H %an %ar %s" --graph

reset the last commit, but before a push

git reset HEAD^

revert to origin, destroying local changes and files in conflict

git reset --hard origin/branchname

add a annotated tag

git tag - a name

zip the files that had changed

git archive --format zip -o changed.zip HEAD `git diff --name-only HASH`

ref: (from http://www.ls12style.co.uk/blog/2011/09/git-tip-creating-zip-of-changed-files/)

listing remotes

git remote -v

Adding a remote

git remote add REMOTE_NAME REMOTE_URL

update a fork with the upstream (need the upstream to had been added)

git fetch upstream
git checkout BRANCH
git merge upstream/BRANCH report with number of commits per author

git shortlog -s -n --all --no-merges
