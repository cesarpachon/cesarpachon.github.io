JSON processing with jq

just pretty-print:

cat file.json | jq

example:

cat sample.json | jq { "a": { "data": "hello" }, "b": { "data": "there" } }

print array of first-level keys:

cat file.json | jq 'keys'

example:

cat sample.json | jq 'keys' [ "a", "b" ]

print only a part of the json:

cat sample.json | jq '.a' { "data": "hello" }

print only the keys of a part of the json:

cat sample.json | jq '.a | keys' [ "data" ]
