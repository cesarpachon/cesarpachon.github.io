VIM

Search

search file by name and edit:

:find filename

Search files that contains expresion, and open the quickfix list to browse the results:

:grep -r expresion :copen :close

Registers

copy or paste into multiple registers:

"{register} followed by y, d, p

markers

  • mk (set marker to k, You can use up to 26 markers a-z
  • ‘k move to the begging of the line of marker location ‘k’
  • `k move to marker ‘k’

macros

q{letter} to enter recording mode.

q to stop recording mode.

{number}@{letter} to execute. number defaults to 1

ex: "qq" starts recording macro named 'q'. "@q" executes it.

others

:set number #enable line numbering

:set tabstop=2 #set tab size

:set nowrap #disable wrapping

#use vi mode in the shell

set -o vi #for bash

bindkey -v #for zshell

vit (Visually select In Tag) #select content inside a tag

%s/foo/bar/g #replace in the whole document

% = move to matching brace

:set mouse:a
