surfraw: Search

it launch a browser. if you don't want a graphical browser, define these variables (with export, by example): SURFRAW_text_browser=/usr/bin/lynx SURFRAW_graphical=no from here: http://askubuntu.com/questions/29540/browsing-the-internet-from-the-command-line Here's some example usage. To search Google for "nm-applet memory leak" : sr google nm-applet memory leak To search for an RFC dealing with S/MIME: sr rfc s/mime Translate a word: sr translate logiciel Find torrents: sr piratebay natty narwhal
